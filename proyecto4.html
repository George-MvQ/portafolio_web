Proyecto C#

Descripcion:

En este proyecto, exploraremos la implementación de una solución de bitacorización en SQL Server que permitirá el monitoreo detallado de las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) realizadas por los usuarios en las tablas de una base de datos. Utilizaremos C# para desarrollar un pequeño programa que demuestre el funcionamiento de esta solución, permitiendo a los usuarios interactuar con las tablas y observar cómo se registran las acciones en la bitácora.

La bitacorización es fundamental para mantener la seguridad y la trazabilidad de las acciones realizadas en una base de datos. En este proyecto, implementaremos una solución que rastrea las operaciones CRUD, creando un registro de cada acción para fines de auditoría y seguimiento.

descripcion del proyecto:

  1. Configuración de Bitacorización

    Implementaremos una opción para activar y desactivar la bitacorización para una tabla específica. Cuando esté activada, se registrarán las acciones CRUD realizadas por los usuarios.

  2. Implementación de Triggers:

    Uso triggers en SQL Server para capturar las operaciones CRUD. Los triggers se activarán automáticamente cuando ocurran acciones en la tabla y registrarán los detalles de la acción en una tabla de bitácora.

  3. Procedimientos Almacenados:

    Crearemos procedimientos almacenados para realizar operaciones CRUD en las tablas. Estos procedimientos serán utilizados por la interfaz de usuario en C# para interactuar con la base de datos.

  4. Registro de Bitácora:

    Cada vez que un usuario realice una operación CRUD, se registrará en una tabla de bitácora, incluyendo el tipo de acción, la fecha y hora, el usuario responsable y detalles adicionales.

Tecnologi usada

  • C#

  • SQL SERVER
